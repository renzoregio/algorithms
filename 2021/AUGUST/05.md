### QUESTION

#### [1556. Thousand Separator](https://leetcode.com/problems/thousand-separator/)

Given an integer n, add a dot (".") as the thousands separator and return it in string format.

```
Example 1:
Input: n = 987
Output: "987"

Example 2:
Input: n = 1234
Output: "1.234"

Example 3:
Input: n = 123456789
Output: "123.456.789"

Example 4:
Input: n = 0
Output: "0"
```

Constraints:

* 0 <= n < 2<sup>31</sup>

-----

### Python

```py
class Solution(object):
    def thousandSeparator(self, n):
        """
        :type n: int
        :rtype: str
        """
        n = str(n)[::-1]
        num_count = 0
        num_string = ""
        for num in n:
            if num_count > 0 and num_count % 3 == 0:
                num_string += "."
                num_string += num
            else:
                num_string += num
            num_count += 1
        return num_string[::-1]     
```
